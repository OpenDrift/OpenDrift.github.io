class opendrift.readers.reader_netCDF_CF_generic.Reader(filename=None, name=None, proj4=None)[source]¶

Bases: opendrift.readers.basereader.BaseReader

A reader for CF-compliant netCDF files. It can take a single file, or a file pattern.

Args:
param filename

A single netCDF file, or a pattern of files. The netCDF file can also be an URL to an OPeNDAP server.

type filename

string, requiered.

param name

Name of reader

type name

string, optional

param proj4

PROJ.4 string describing projection of data.

type proj4

string, optional

Example:

from opendrift.readers import Reader
r = Reader("arome_subset_16Nov2015.nc")

Several files can be specified by using a pattern:

from opendrift.readers import Reader
r = Reader("*.nc")

or an OPeNDAP URL can be used:

from opendrift.readers import Reader
r = Reader('http://thredds.met.no/thredds/dodsC/meps25files/meps_det_extracted_2_5km_latest.nc')
